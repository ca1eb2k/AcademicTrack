# AcademicTracker - Python GUI Applications

This repository contains a collection of Python GUI applications developed using the Tkinter library. These applications are part of the AcademicTracker suite and are designed to help students manage their academic tasks and information.

## Applications

### Homework Journal
The Homework Journal application allows users to record and manage their daily homework tasks. Users can enter homework details for each subject and navigate between different dates to track their homework progress.

### Rank Score Calculator
The Rank Score Calculator application helps users calculate their academic rank scores based on achieved, merit, and excellence grades. Users can input their grades, and the application calculates the total rank score.

### Timetable
The Timetable application assists users in managing their daily schedules. Users can input their class or activity schedule for each day, and the application displays a dynamic timetable for the week.


## Usage

1. Ensure you have Python installed on your system.
2. Clone this repository or download the required files.
3. Install the necessary libraries using the command: `pip install tk`.

Run the Python login file using the command: `python login.py`. Make sure to have the required icons and data storage files in the same directory.
